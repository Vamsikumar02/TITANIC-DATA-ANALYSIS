# Titanic Data Analysis
📖 Overview

This project presents an Exploratory Data Analysis (EDA) of the Titanic dataset. The goal is to analyze passenger demographics, ticket details, and travel information to identify factors that influenced survival during the Titanic disaster. The analysis is performed using Python and popular data science libraries. The Titanic dataset is a widely used benchmark for learning EDA and predictive analytics.

# 🎯 Objectives 

Understand the dataset structure and features
Perform data cleaning and handle missing values
Explore passenger demographics and travel patterns
Analyze survival trends across different groups
Visualize relationships between variables.

# 🔍 Key Insights
Female passengers had a significantly higher survival rate.
First-class passengers were more likely to survive than passengers in lower classes.
Passenger class and fare showed strong relationships with survival outcomes.
Age and family-related features provided additional insights into survival patterns.
